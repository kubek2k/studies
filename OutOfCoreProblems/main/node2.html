Variety of OOC problems

There are many kinds of OOC problems, some of them are:
  • Big matrices operations ( inverting, multiplying, transpositioning etc. )
  • Huge sets of equations solving ( also decompositions )
  • Processing great meshes ( rendering etc. )
  • Data mining
  • Out of core sorting
  • Markov models out of core symbolic solving
